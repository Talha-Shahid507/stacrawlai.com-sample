StackCrawlAI Documentation

Welcome to StackCrawlAI — the AI-powered web crawler built for LLMs, RAG pipelines, and agentic workflows. This guide covers everything from your first API call to advanced multi-page crawls.

✦ What's new in v3.0

Agentic Crawler with adaptive stopping, Prefetch Mode (10× faster URL discovery), crash-recovery checkpoints, and new stealth fingerprinting engine.

Quick Start

Get your first crawl running in under 2 minutes:

quickstart.py
from stackcrawlai import Crawler

crawler = Crawler(api_key="sk-your-key-here")

result = crawler.extract(
  url="https://news.ycombinator.com",
  prompt="Extract all post titles, scores, and author names"
)

print(result.data) # structured JSON ✓

Installation

Install the Python SDK via pip:

terminal
$ pip install stackcrawlai
# or for Node.js:
$ npm install @stackcrawlai/sdk

Authentication

All API requests require an API key. You can generate one in your Dashboard → API Keys.

Include it in the Authorization header or pass it to the SDK constructor:

HTTP header
Authorization: Bearer sk-your-api-key

Crawl Endpoint

The crawl endpoint follows links from a starting URL and processes the entire site or a section of it.

POST /v1/crawl Start a crawl job
GET /v1/crawl/{job_id} Get job status & results
DELETE /v1/crawl/{job_id} Cancel a crawl job

Parameters

Parameter Type Description
url string* Starting URL for the crawl
depth integer Max crawl depth (default: 3)
max_pages integer Max pages to crawl (default: 1000)
output string "markdown" | "json" | "html"
stealth boolean Enable stealth mode (Pro+)

Python SDK

The Python SDK wraps the REST API with async support, automatic retries, and Pydantic models.

async_example.py
import asyncio
from stackcrawlai import AsyncCrawler

async def main():
  async with AsyncCrawler(api_key="sk-...") as crawler:
    job = await crawler.crawl(
      url="https://docs.example.com",
      depth=4, stealth=True
    )
    # Stream results as they come in
    async for page in job.stream():
      print(page.url, f"→ {len(page.markdown)} chars")

asyncio.run(main())
